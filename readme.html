ก่อนจะเริ่มต้น

ขอกล่าวยินดีต้อนรับ WordPress เป็นโปรเจคที่พิเศษมากสำหรับผม นักพัฒนาและผู้สนับสนุนทุกคนต่างได้สรรค์สร้างรวบรวมความเป็นเลิศเข้าไว้ด้วยกัน พวกเราได้ร่วมกันสร้างบางสิ่งที่สวยงาม ผมภูมิใจที่ได้เป็นส่วนหนึ่ง เวลากว่าหลายพันชั่วโมงที่ทุ่มเทให้กับ WordPress เรามุ่งมั่นที่จะทำให้มันดีขึ้นในทุกๆวัน ขอบคุณที่เลือกให้ WordPress เป็นส่วนหนึ่งของชีวิตคุณ

— Matt Mullenweg

ขั้นตอนติดตั้ง: การติดตั้งเพียง 5 นาที ที่มีชื่อเสียง

  1. แตกซิปไฟล์ (Unzip) ไว้ที่ไดเรกทอรีว่าง และอัปโหลดไฟล์ทั้งหมดไปวางยังที่ที่ต้องการติดตั้ง
  2. เปิด wp-admin/install.php ด้วยเบราว์เซอร์ เพื่อเข้าสู่ขั้นตอนการตั้งค่าการเชื่อมต่อฐานข้อมูลในไฟล์ wp-config.php
    1. การทำงานนี้ไม่สามารถใช้ได้กับทุกเว็บโฮส ดังนั้นไม่ต้องกังวล หากคุณทำไม่ได้ ให้เปิด wp-config-sample.php ด้วยโปรแกรมที่ใช้สำหรับสร้างและแก้ไขข้อความ (Text editor) แทน เช่น WordPad หรือโปรแกรมอื่นที่คล้ายกัน และใส่ค่าการเชื่อมต่อฐานข้อมูลของคุณ
    2. บันทึกไฟล์เป็นชื่อ wp-config.php และอัปโหลด
    3. เปิด wp-admin/install.php ด้วยเบราว์เซอร์
  3. เมื่อตั้งค่าการเชื่อมต่อฐานข้อมูลในไฟล์เสร็จเรียบร้อย ตัวติดตั้งจะสร้างตารางฐานข้อมูลสำหรับบล็อกของคุณ ถ้ามีความผิดพลาดเกิดขึ้น ให้ตรวจสอบไฟล์ wp-config.php และลองติดตั้งใหม่อีกครั้ง หากยังคงติดตั้งไม่สำเร็จ แนะนำให้ไปยัง Support Forums พร้อมด้วยข้อมูลทั้งหมดที่คุณมี
  4. ถ้าคุณไม่ได้กำหนดรหัสผ่านเอง ให้ใช้รหัสผ่านที่ได้รับจากระบบ และถ้าคุณไม่ได้กำหนดชื่อผู้ใช้อีกด้วย ชื่อผู้ใช้ที่ได้รับจะเป็น admin
  5. ตัวติดตั้งจะนำคุณไปที่ หน้าเข้าสู่ระบบ คุณสามารถเข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่านที่กำหนดไว้ระหว่างการติดตั้ง ถ้ารหัสผ่านถูกสร้างขึ้นจากระบบ คุณสามารถเปลี่ยนแปลงได้ที่ “หน้าข้อมูลส่วนตัว”

การอัปเดต

การอัปเดตอัตโนมัติ

คุณสามารถอัปเดตอัตโนมัติได้ หากเป็นการอัปเดตของรุ่น 2.7 หรือสูงกว่า:

  1. เปิด wp-admin/update-core.php ด้วยเบราว์เซอร์ และทำตามคำแนะนำ
  2. ต้องการขั้นตอนมากกว่านี้เหรอ? ไม่ละ มีขั้นตอนเท่านี้เอง!

การอัปเดตด้วยตัวเอง

  1. สำเนาไฟล์ทั้งหมดที่มีการปรับแก้ เช่น ไฟล์ index.php ให้เรียบร้อยก่อนที่จะอัปเดต
  2. ลบไฟล์ทั้งหมดของ WordPress รุ่นเก่า เก็บไฟล์ที่คุณปรับแก้ไว้
  3. อัปโหลดไฟล์ WordPress รุ่นใหม่
  4. เปิด /wp-admin/upgrade.php ด้วยเบราว์เซอร์

การย้ายมาจากระบบอื่น

คุณสามารถ นำเข้าข้อมูลจากระบบอื่นๆ มายัง WordPress ได้ โดยขั้นตอนแรกคือ ติดตั้ง WordPress ให้เรียบร้อยตามวิธีการข้างต้น หลังจากนั้นให้ย้ายข้อมูลด้วย เครื่องมือการนำเข้า ของเรา

คุณสมบัติที่จำเป็นของระบบ

  • PHP รุ่น 5.2.4 หรือสูงกว่า
  • MySQL รุ่น 5.0 หรือสูงกว่า

คุณสมบัติที่แนะนำ

  • PHP รุ่น 7 หรือสูงกว่า
  • MySQL รุ่น 5.6 หรือสูงกว่า
  • โมดูล mod_rewrite ของ Apache
  • รองรับ HTTPS
  • การลิงก์ไปที่ wordpress.org จากเว็บไซต์ของคุณ

แหล่งข้อมูลออนไลน์

หากคุณมีคำถามอื่นๆที่เอกสารฉบับนี้ไม่ได้กล่าวถึง ขอให้ลองไปที่แหล่งข้อมูลออนไลน์ของ WordPress ดังนี้:

The WordPress Codex
Codex เป็นสารานุกรมของ WordPress มีเนื้อหาครอบคลุม WordPress มากที่สุด เท่าที่จะหาได้
The WordPress Blog
บล็อกของ WordPress เป็นที่ที่คุณจะทราบข่าวการอัปเดตรุ่นล่าสุด และข่าวอื่นๆเกี่ยวกับ WordPress โดยใน WordPress รุ่นหลังๆจะมีการแสดงข่าวของ WordPress ที่หน้าควบคุมอีกด้วย
WordPress Planet
WordPress Planet เป็นผู้ที่รวบรวมเรื่องต่างๆจากเว็บบล็อก WordPress ทั่วโลก
WordPress Support Forums
ถ้าคุณลองค้นหาจากทุกที่แล้ว แต่ยังไม่สามารถหาคำตอบได้ ชุมชนช่วยเหลือของ WordPress (WordPress Support Forums) นั้นเป็นชุมชนที่มีความกระตือรือร้นมาก และมีขนาดใหญ่ พร้อมที่จะให้ความช่วยเหลือ เพื่อให้พวกเขาช่วยเหลือคุณได้ คุณควรจะบรรยายชื่อหัวข้อ และอธิบายคำถามของคุณให้ละเอียดที่สุดเท่าที่คุณจะทำได้
WordPress IRC Channel
IRC ของ WordPress เป็นช่องทางการคุยออนไลน์ ที่ใช้สำหรับแลกเปลี่ยนความคิดเห็นของกลุ่มผู้ใช้ WordPress และช่วยเหลือกันในบางหัวข้อ The above wiki page should point you in the right direction. (irc.freenode.net #wordpress)

สุดท้ายนี้

  • หากคุณมีข้อเสนอแนะ ไอเดีย หรือความคิดเห็น หรือคุณอาจจะพบข้อผิดพลาด! กรุณาเข้ามาคุยกับเราที่ Support Forums
  • WordPress มีระบบ API ที่มีความสามารถมากมายสำหรับปลั๊กอิน ช่วยให้การเขียนโค้ดเป็นเรื่องง่าย ถ้าคุณเป็นผู้พัฒนาและสนใจ WordPress คุณไม่ควรที่จะแก้ไขโค้ดของ Core กรุณาดูข้อมูล Plugin Developer Handbook

ส่งต่อความรักที่คุณมีกับ WordPress

WordPress ไม่มีเงินหลายล้านดอลล่าร์สำหรับการทำแคมเปญการตลาด หรือมีผู้สนับสนุนที่มีชื่อเสียง แต่เรามีบางอย่างที่มีค่ามากกว่านั้น นั่นคือ — คุณ ถ้าคุณชื่นชอบใน WordPress กรุณาบอกต่อไปยังเพื่อนคุณ ช่วยเหลือคนที่มีความรู้น้อยกว่าเพื่อเริ่มต้นการใช้งาน หรือเขียนบทความเกี่ยวกับสื่อที่มองข้ามเรา

WordPress นั้นพัฒนาอย่างเป็นทางการมาจาก b2/cafélog ซึ่งริเริ่มโดย Michel V. งานชิ้นนี้ได้ถูกสืบทอดต่อโดย ผู้พัฒนา WordPress คุณสามารถสนับสนุน WordPress ได้ ผ่านช่องทาง การบริจาค
